# YaaDjango
First step to Django.

## Setup 
### Virtual environment
```> python -m venv venv```

from shell:
```$ source venv/bin/activate```

from powershell:
```>  .\venv\Scripts\Activate.ps1```

from cmd:
```> venv\Scripts\activate.bat```

### Install requirements
```> pip install -r requirements.txt```
